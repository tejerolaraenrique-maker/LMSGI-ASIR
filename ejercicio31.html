Introducción

MySQL es un sistema de gestión de bases de datos relacional (RDBMS) de código abierto que utiliza el lenguaje SQL (Structured Query Language) para gestionar y manipular bases de datos. Fue desarrollado por MySQL AB, una empresa sueca, y ahora es propiedad de Oracle Corporation. MySQL es ampliamente utilizado en aplicaciones web y es conocido por su rendimiento, fiabilidad y facilidad de uso.

show database

El comando SHOW DATABASES; se utiliza para mostrar todas las bases de datos en el servidor MySQL. Este comando es útil para obtener una lista de las bases de datos disponibles y verificar si se ha creado una base de datos específica.

Creacion de una tabla y mostrar sus campos

Para crear una tabla en MySQL, se utiliza el comando CREATE TABLE. Este comando permite definir la estructura de la tabla, incluyendo los nombres y tipos de datos de las columnas. Una vez creada la tabla, se puede utilizar el comando DESCRIBE para mostrar sus campos y tipos de datos.

Carga de registros a una tabla y su recuperación

Para cargar registros en una tabla, se utiliza el comando INSERT INTO. Este comando permite agregar nuevos registros a la tabla especificando los valores para cada columna. Para recuperar registros, se utiliza el comando SELECT, que permite consultar y mostrar los datos almacenados en la tabla.
