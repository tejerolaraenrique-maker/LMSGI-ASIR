HTML

HTML hiper text markup language

El lenguaje HTML es un lenguaje de marcado que se utiliza para estructurar y presentar contenido en la web. Es muy facil ver el codigo HTML en la consola del navigator.

Historia de HTML

El HTML fue creado por Tim Berners-Lee en 1991. Desde entonces, ha evolucionado a través de varias versiones, siendo HTML5 la más reciente y ampliamente utilizada en la actualidad.
